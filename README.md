**Student Management System – Java (Console Based)**

A simple Java console application that allows users to manage student records efficiently.
This project demonstrates Core Java, OOP concepts, ArrayList, and basic CRUD operations — suitable for beginners and Java fresher placements.

 **Features**

 Add Student

 View All Students

 Search Student by ID

 Update Student Details

 Delete Student

 Uses ArrayList for data storage

 Beginner-friendly, easy to understand

 **Technologies Used**

Java (Core Java)

OOP (Classes, Objects, Methods)

ArrayList

Scanner (User Input)

 **Project Structure**
StudentManagement/
│
├── Main.java

 **Code Overview**
# Student.java

Model class

Stores: id, name, age, course

## StudentService.java

Contains business logic

CRUD operations (Add, View, Search, Update, Delete)

## Main.java

User interaction

Menu-driven console application

 **How to Run**

Install JDK 8+

Save all files inside a folder named StudentManagement

Open terminal / command prompt:

javac *.java
java Main
